Introduction to HTML

What is HTML?

HTML (HyperText Markup Language) is the standard language used to structure web pages. It allows you to organize content such as text, images, links, and more, using various tags (elements).

                <p>This is a simple webpage.</p>
            </body>
        </html>

Common HTML Elements

  • Headings: <h1> to <h6>
  • Paragraphs: <p> for text content
  • Links: <a href="url"> for hyperlinks
  • Images: <img src="image.jpg"> to display images
  • Lists: <ul> for unordered lists and <ol> for ordered lists

HTML in Web Development

HTML is the foundation of web development, but it works with:

  • CSS for design and layout
  • JavaScript for interactivity
